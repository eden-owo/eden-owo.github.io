GraphPlan-LLM

語言模型輔助圖結構規劃框架（Scene Graph + LLM）

  • 自然語言 Prompt 快速生成結構化平面場景規劃
  • Scene Graph 管理空間關係，確保設計合理
  • 自動佈局演算法 → 根據約束條件優化物件位置與尺寸
  • 視覺化輸出 → 直接生成可視化平面圖，減少手動繪製時間

專案背景

建築設計師畫平面圖十分耗時，空間規劃和調整需要花費大量時間。

雖然 CAD 很精確，但每個元素都要手動畫，又慢又繁瑣。

設計師藉由 GraphPlan-LLM 輸入文字需求，系統自動生成平面圖，大幅節省初期設計的時間。

核心挑戰

  • 如何確保 LLM 生成的房間節點合理分布在用戶提供的格局邊界內
  • 房間位置需要貼合牆壁，並避免阻擋門口等關鍵區域
  • 相鄰房間的邊界需要對齊，消除縫隙並創建共享牆壁

技術架構：多階段對齊與優化流程

  • 1️⃣ LLM 圖結構生成
    LLM 生成房間類型和連接關係（如「客廳-主臥」），建立空間拓撲圖。
  • 2️⃣ 力導向佈局演算法
    使用 Fruchterman-Reingold 演算法模擬排斥力（避免重疊）、吸引力（相鄰房間靠近）與邊界約束力，經 100 次迭代達到物理平衡。
  • 3️⃣ 深度學習模型優化
    神經網路模型根據節點位置、房間類型和邊界約束，生成符合建築規範的房間框（bounding boxes）。
  • 4️⃣ 幾何對齊與正規化
    邊界吸附（< 18 單位）、門口避讓（Shapely 碰撞檢測）、鄰居對齊（消除縫隙）、生成最終多邊形。
